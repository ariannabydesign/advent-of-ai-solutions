## Matching Process

### When Someone Claims an Item

**Golden Rule: They describe it BEFORE you show it!**

```
1. ASK: "Can you describe the item you lost?"
2. VERIFY: Does description match? (color, brand, contents)
3. CHECK ID: Required for high-value items
4. LOG: Record the claim (successful or not)
5. RELEASE: Have them sign for the item
```

### Why We Verify First

- Prevents false claims
- Protects genuine owners
- Creates paper trail
- Reduces disputes

### High-Value Item Release

For phones, wallets, jewelry, or items worth $50+:

- [ ] Photo ID required
- [ ] ID matches name on item (if visible)
- [ ] Claimant signs release form
- [ ] Note ID number in log

## Common Scenarios

### "I lost my phone!"

1. Ask: What kind? Color? Case? Lock screen image?
2. Check food court first (most common)
3. If not found, take their contact info
4. Call them if it turns up

### "My kid lost their stuffed animal!" ğŸ˜¢

1. This is **URGENT** - prioritize!
2. Get detailed description (name, color, size, unique features)
3. Check kids areas immediately
4. Radio all staff to check their areas
5. If not found, take contact info and keep looking

### "I think someone stole it!"

1. Don't accuse or assume theft
2. Say: "Let's check Lost & Found first"
3. If not there, refer to Security
4. Security handles theft investigations

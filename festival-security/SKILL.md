## Emergency Codes

**Memorize these - they save lives!**

| Code | Situation | Immediate Actions |
|------|-----------|-------------------|
| üü° **Code Yellow** | Lost Child | All exits notify, broadcast description, security converges |
| üîµ **Code Blue** | Medical Emergency | Clear path to medical tent, call 911, keep crowd back |
| ‚ö™ **Code White** | Weather Emergency | Announce shelter locations, secure loose items, guide guests |
| üü† **Code Orange** | Suspicious Activity | Security team converges, **DO NOT confront alone** |

### Code Yellow - Lost Child Protocol

1. Get description: name, age, clothing, last seen location
2. Radio all exits immediately
3. Broadcast description over PA
4. Security forms search grid from last known location
5. Notify Festival Director if not found in 10 minutes

### Code Blue - Medical Emergency Protocol

1. Call 911 immediately
2. Radio medical tent for on-site response
3. Clear path from medical tent to patient location
4. Keep crowd back - designate someone to guide EMS in
5. Document everything for incident report

### Code White - Weather Emergency Protocol

1. Monitor weather alerts continuously
2. Announce shelter locations (3 times minimum)
3. Secure loose items, shut down outdoor attractions
4. Guide guests to shelter areas
5. Do NOT resume until all-clear from Festival Director

### Code Orange - Suspicious Activity Protocol

1. **DO NOT CONFRONT ALONE** - This is critical
2. Radio security team with location and description
3. Security team converges and assesses
4. Maintain visual if safe to do so
5. Festival Director decides on escalation (police, etc.)

### Noise Complaint Escalation

This is a **3-strike system**:

```
Strike 1: VERBAL WARNING
   ‚Üí Politely ask them to lower volume
   ‚Üí Document time and conversation

Strike 2: WRITTEN WARNING  
   ‚Üí Formal written notice
   ‚Üí Explain next step is shutdown
   ‚Üí Both parties sign

Strike 3: SHUTDOWN
   ‚Üí Vendor must cease music/noise immediately
   ‚Üí May continue selling quietly
   ‚Üí Note in vendor file for future events
```

### Vendor Disputes

**Never take sides publicly!**

1. Separate the parties
2. Move discussion to back office
3. Hear both sides privately
4. Mediate a solution
5. Document outcome
6. Follow up with both parties later

### Payment Issues

- **Rule: 50% upfront, balance day-of**
- No exceptions without Festival Director approval
- Cash or card accepted
- Provide receipt for all payments
- Chase outstanding balances BEFORE next event
